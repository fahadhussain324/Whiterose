Why WhiteRose?

Built by Biomedical Scientists in the core Blood Sciences disciplines, WhiteRose LIMS aims to work for the laboratory staff, creating a core engine integrating the laboratory to the rest of the hospital ecosystem.

Core Development

WhiteRose v1 is built on Java and Javascript, with MySQL database fueling the backend. It has a custom built engine designed for reliability, scalability and easy to integrate interface for seamless implementation and daily use.
